Amazon Free Books WS API


This tutorial will explain you how to use on amazon free books API.

There is 3 methods to use :


1. /getAllFree
2. /getFreeBooksByName
3. /getFreeBooksByDates

Follow the instructions !


Instructions:

Verb:

GET

Path:

https://freebooksws.herokuapp.com/getAllFree

Get free books by dates

Verb:

POST

Path:

https://freebooksws.herokuapp.com/getFreeBooksByDates

Description:

You need to send 2 values, the first will be startDate and the second will be lastDate.
     All the books are free for only one month !
     Exmple: startDate: March, lastDate:April .
